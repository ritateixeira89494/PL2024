# TPC1

Funções realizadas:
- definir_dicionario(): esta função trata de definir cada atleta no seu próprio dicionário para depois obter um dicionário de dicionários para poder utilizar nos exercícios pedidos
- ex1(): trata de fazer uma lista ordenada alfabeticamente de acordo com as modalidades de cada atleta
- ex2(): apresenta as percentagens de atletas aptos e inaptos para a prática desportiva. Esta percentagem foi decidida através da coluna 'resultado' (true->apto; false->inapto)
- ex3(): organização dos atletas por faixa etária (intervalos de 5 anos)
